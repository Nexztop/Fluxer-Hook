Markdown & LaTeX Guide

Headers & Formatting

Heading 1

Heading 2

Heading 3
**bold** Bold
*italic* Italic
__underlined__ Underlined
~~strike~~ Strikethrough
||spoiler|| Hidden Spoiler
`code` Inline Code
-# text Small subtext
Block Elements & Admonitions
> text Single line quote
>>> text Multiline quote
> [!NOTE]
Text		 Note
> [!TIP]
Text		 Tip
> [!IMPORTANT]
Text		 Important
> [!WARNING]
Text		 Warning
> [!CAUTION]
Text		 Caution
LaTeX (Math)
Fluxer supports complex LaTeX math formulas. Wrap them in a code block with `latex`.
```latex
\int_0^\infty e^{-x} dx
```
